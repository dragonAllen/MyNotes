# 计算机网络

## 概述

### 网络类别

1. 广域网 WAN 
2. 城域网 MAN
3. 局域网 LAN
4. 个人区域网 PAN/WPAN



### 性能指标

1. 速率

指连接在计算机网络上的主机在数字信道上传送数据的速率;

单位为 b/s(bit/second),kb/s(k=10的三次方=千),Mb/s(M=10的六次方=兆),Gb/s(G=10的九次方),Tb/s(T=10的十二次方=太);
